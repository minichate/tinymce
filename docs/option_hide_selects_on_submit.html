Option: hide_selects_on_submit

This option enables you to control if select boxes should be disabled before a submit event occurs. This option is set to false by default since most applications doesn't break if there is some extra form data posted together with TinyMCE but this option might come in handy for some applications that map form fields to database fields.

Example of usage of the hide_selects_on_submit option:

tinyMCE.init({
	...
	hide_selects_on_submit : true
});
